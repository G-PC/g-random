# 🎲 G-RANDOM / G-РАНДОМ

![Версия](https://img.shields.io/badge/version-1.2.0-green.svg)
![Лицензия](https://img.shields.io/badge/license-MIT-blue.svg)

**G-RANDOM** — это набор инструментов для случайных выборов. Идеальное решение для игр (Мафия, Крокодил), розыгрышей и бытовых решений.

## ✨ Основные инструменты
*   **🎭 Рандомайзер ролей**: Распределение ролей для игры в Мафию по настроенной таблице баланса.
*   **🎡 Колесо фортуны**: Случайный выбор слова или имени из списка.
*   **🔄 Колесо на выбывание**: Случайный выбор с удалением выпавшего варианта.
*   **🎲 Генератор чисел**: Генерация случайных чисел в заданном диапазоне.
*   **⚙️ Дополнительное**: Описание ролей и форма обратной связи.

## 🚀 Быстрый старт

Вы можете использовать проект онлайн или установить как приложение.

### 🌐 Онлайн версия
Просто перейдите по ссылке: [https://g-wpi.github.io/g-random/](https://g-wpi.github.io/g-random/)

### 📱 Локальная установка (PWA)
Сайт можно установить как приложение на телефон:
*   **На Android**: Откройте сайт в Chrome → нажмите "Установить приложение".
*   **На iPhone (iOS)**: Откройте в Safari → нажмите "Поделиться" → "На экран «Домой»".

### ⬇️ Установка APK
Для Android доступен файл для прямой установки: `G-RANDOM_android.apk`

## 📂 Структура проекта
*   `index.html` — Главное меню.
*   `detect.html` — Определение устройства и роутинг.
*   `add.html` / `m-add.html` — Страница дополнительных материалов.
*   `numbers.html` / `m-numbers.html` — Генератор чисел.
*   `fortune.html` / `m-fortune.html` — Колесо фортуны.
*   `elimination.html` / `m-elimination.html` — Колесо на выбывание.
*   `roles.html` / `m-roles.html` — Рандомайзер ролей.
